# EU Parliament Representatives Information Repository
This repository contains Excel files with contact information for Members of the European Parliament (MEPs), including official email addresses.

You are welcome to download and use these files to contact EU representatives regarding the Iran revolution and to urge them to designate the IRGC as a terrorist organization.

Currently, this repository includes information on MEPs from the following countries:
- France (`france.xlsx`)
- Italy (`italy.xlsx`)
- Spain (`spain.xlsx`)
